CODEBEAT

Introduction

Codebeat is an audio programming library designed for music composition and deconstruction. A combination of efficient syntax and data-driven analysis, Codebeat has an answer for every musical quandry. Because Codebeat is built on JavaScript, it can be integrated painlessly on the client-side or server-side of any web application. Take a look at the samples below or write you own. More documentation is available on the project's GitHub page.

These sample melodies are intended to demonstrate the ease and power of the Codebeat syntax. More extensive compositions accompanied by data visualizations are coming soon. Remember to try it for yourself.
